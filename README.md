# ChiTalk: Smart Location-Based Translator

ChiTalk adalah aplikasi penerjemah pintar berbasis Android yang memanfaatkan GPS untuk mendeteksi lokasi pengguna dan menerjemahkan teks ke bahasa lokal secara otomatis dengan bantuan **Google Gemini AI**.

---

## Fitur Utama
* **Deteksi Lokasi Otomatis:** Menggunakan *Fused Location Provider* untuk mendeteksi kode negara pengguna secara *real-time*.
* **AI Translation:** Terintegrasi dengan model **Gemini 2.5 Flash** via API untuk menghasilkan terjemahan teks yang cepat, akurat, dan natural.
* **Tema "Soft Blush":** Desain antarmuka (UI/UX) yang minimalis, *clean*, dan elegan menggunakan perpaduan warna pastel pink dan kartu putih (*CardView*).

## Teknologi yang Digunakan
* **Bahasa Pemrograman:** Java
* **IDE:** Android Studio
* **API:** Google Gemini API (Generative Language)
* **Library Jaringan:** `OkHttp3`
* **Layanan Lokasi:** Fused Location Provider (Google Play Services)

---

## Cara Menjalankan Aplikasi
1. *Clone repository* ini ke Android Studio.
2. Buka file `MainActivity.java`.
3. Masukkan Kunci API Gemini milikmu pada variabel `API_KEY`.
4. Jalankan (*Run*) aplikasi pada perangkat atau emulator.
5. Izinkan akses lokasi dan pastikan GPS perangkat dalam keadaan aktif.
